# Joint Dictionary Learning + SVM FYP

This repository scaffolds the implementation for a joint dictionary learning + linear SVM model on binary image-classification tasks, with MNIST `3 vs 8` as the default starting task.

## Layout

- `src/`: reusable implementation modules
- `notebooks/`: experiment notebooks
- `requirements.txt`: Python dependencies

## First setup

```bash
python3 -m pip install -r requirements.txt
```

Then start with `notebooks/01_problem_setup.ipynb`.
